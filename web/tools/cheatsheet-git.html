DevTools

Git Commands Cheatsheet

Init
git init
Initialize repository
Clone
git clone URL
Clone remote repo
Status
git status
Changes summary
Add
git add .
Stage all
Commit
git commit -m "msg"
Commit staged
Branch
git branch
List branches
Checkout
git checkout -b feat
New branch
Merge
git merge feat
Merge branch
Pull
git pull
Fetch + merge
Push
git push origin main
Push branch
Stash
git stash
Save WIP
Rebase
git rebase main
Reapply commits
Reset
git reset --hard HEAD~1
Drop last commit
Revert
git revert
New inverse commit
Log
git log --oneline --graph
Compact history
