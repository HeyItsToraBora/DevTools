DevTools

Linux Commands Cheatsheet

List
ls -la
List files
Change dir
cd /path
Go to folder
Copy
cp src dst
Copy file
Move
mv src dst
Move/rename
Remove
rm -rf path
Delete file/dir
Find
find . -name "*.log"
Search files
Grep
grep -R "text" .
Search content
Tar
tar -czf out.tgz dir/
Create archive
Curl
curl -sSL URL
HTTP request
System
top, df -h, du -sh *
Monitor/disk usage
Perms
chmod 755 file
Change mode
SSH
ssh user@host
Connect via SSH
